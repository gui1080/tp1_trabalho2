Smoke Test

Trabalho 2 de TP1 por Guilherme Braga e Gabriel Matheus


1º Teste

Propósito:

Testar do ponto de vista de um novo usuário.

Procedimento:
  1. Executar o Programa
  2. Criar conta
  3. Fechar o Programa

2º Teste

Propósito:

Testar do ponto de vista de um usuário recorrente.

Procedimento:
  1. Executar o Programa
  2. Fazer login com sua conta que foi criada antes
  3. Fechar o Programa

3º Teste

Propósito:

Testar do ponto de vista de um usuário casual, pesquisando possiveis eventos.

Procedimento:
  1. Executar o Programa
  2. Clicar no local relacionado aos Eventos
  3. Checar os Eventos existentes, com suas Apresentações
  4. Fechar o Programa

4º Teste

Propósito:

Testar do ponto de vista do dono de um Evento.

Procedimento:
  1. Executar o Programa
  2. Fazer login
  3. Criar um novo Evento, informando todos os seus dados relacionados
  4. Fechar o Programa
