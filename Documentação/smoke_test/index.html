Smoke Test

Trabalho 2 de TP1 por Guilherme Braga (17/0162290) e Gabriel Matheus (17/0103498)


1º Teste

Propósito:

Testar que o programa foi compilado com sucesso.

Procedimento:
  1. Executar o Programa
  2. Fechar o programa usando a interação com o menu que foi criado (interação via cin e cout)


2º Teste

Propósito:

Testar do ponto de vista de um novo usuário, ver a criação de um novo usuário.

Procedimento:
  1. Executar o Programa
  2. Criar conta
  3. Fechar o Programa

3º Teste

Propósito:

Testar do ponto de vista de um usuário recorrente, ver a validação de um usuário recorrente.

Procedimento:
  1. Executar o Programa
  2. Criar conta
  3. Sair da conta
  4. Fazer login com sua conta que foi criada antes
  5. Fechar o Programa

4º Teste

Propósito:

Testar do ponto de vista de um usuário casual, testando a possibilidade de pesquisar possíveis Eventos.

Procedimento:
  1. Executar o Programa
  2. Selecionar a opção relacionada aos Eventos
  3. Ver os Eventos existentes
  4. Fechar o Programa

5º Teste

Propósito:

Testar do ponto de vista do dono de um Evento, validar a criação de Evento.

Procedimento:
  1. Executar o Programa
  2. Entrar no sistema com seu usuário
  3. Criar um novo Evento, informando todos os seus dados relacionados
  4. Fechar o Programa

6º Teste

Propósito:

Testar do ponto de vista do vendedor, criando uma apresentação.

Procedimento:
  1. Executar o Programa
  2. Fazer login
  3. Adicionar uma apresentação
  4. Ver sua apresentação como estando cadastrada
  5. Fechar o Programa

7º Teste

Propósito:

Testar do ponto de vista do comprador, descadastrando sua sua conta.

Procedimento:
  1. Executar o Programa
  2. Criar uma conta nova
  3. Entrar no sistema
  4. Efetuar descadastramento
  5. Terminar a operação com sucesso
  6. Fechar o Programa

8º Teste

Propósito:

Testar do ponto de vista do comprador.

Procedimento:
  1. Executar o Programa
  2. Entrar no sistema com a sua conta
  3. Ver os eventos disponíveis e suas respectivas apresentações
  4. Terminar a operação com sucesso
  5. Fechar o Programa

9º Teste

Propósito:

Testar do ponto de vista de quem organiza um determinado Evento, descadastrar um Evento.

Procedimento:
  1. Executar o Programa
  2. Entrar no sistema
  3. Criar um Evento novo
  4. Descadastrar o Evento logo em seguida
  5. Terminar a operação com sucesso
  6. Fechar o Programa

10º Teste

Propósito:

Testar procedimento de busca.

Procedimento:
  1. Executar o Programa
  2. Criar conta nova
  3. Ver eventos disponíveis
  4. Buscar uma apresentação/evento de seu desejo
  5. Obter um resultado que condiz com o esperado
  6. Fechar o Programa
